# turul-a2a-proto

Generated protobuf types for the [A2A Protocol v1.0](https://github.com/a2aproject/A2A)
(package `lf.a2a.v1`), produced via `prost` + `pbjson` from the normative
`proto/a2a.proto`.

This crate is consumed by [`turul-a2a-types`](https://crates.io/crates/turul-a2a-types)
and [`turul-a2a`](https://crates.io/crates/turul-a2a). Most users will interact
with the ergonomic wrappers in `turul-a2a-types` instead.

See the [workspace README](https://github.com/aussierobots/turul-a2a#readme)
for the full project overview, architecture, and examples.
